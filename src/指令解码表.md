# 指令解码表

> 项目覆盖的所有 RISC-V 指令编码速查。按 opcode 排列，列出 funct3/funct7/rs2 等字段的区分方式。

## 总览：7 位 opcode 分发

```
 opcode[6:0]
 ────────────
 0x37  → exec_lui()         U-type   LUI
 0x17  → exec_auipc()       U-type   AUIPC
 0x6F  → exec_jal()         J-type   JAL
 0x67  → exec_jalr()        I-type   JALR
 0x63  → exec_branch()      B-type   BEQ/BNE/BLT/BGE/BLTU/BGEU
 0x03  → exec_load()        I-type   LB/LH/LW/LBU/LHU
 0x23  → exec_store()       S-type   SB/SH/SW
 0x13  → exec_op_imm()      I-type   ADDI/SLTI/SLTIU/XORI/ORI/ANDI/SLLI/SRLI/SRAI
 0x33  → exec_op()          R-type   ADD/SUB/SLL/SLT/SLTU/XOR/SRL/SRA/OR/AND
                     (funct7=0x01 → exec_m_muldiv  M扩展)
 0x73  → exec_system()      I-type   ECALL/EBREAK/MRET
 0x0F  → exec_fence()       I-type   FENCE (NOP)
 ───────────────────────────────────── F 扩展 ─────
 0x07  → exec_load_fp()     I-type   FLW
 0x27  → exec_store_fp()    S-type   FSW
 0x53  → exec_fp_op()       R-type   FADD/FSUB/FMUL/FDIV/... (20 条)
 0x43  → exec_fma()         R4       FMADD.S
 0x47  → exec_fma()         R4       FMSUB.S
 0x4B  → exec_fma()         R4       FNMSUB.S
 0x4F  → exec_fma()         R4       FNMADD.S
```
