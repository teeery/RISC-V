# 缩写约定

本文档统一项目中使用的变量命名缩写，四个模块共用一个标准，避免各写各的。

---

## 通用缩写

| 缩写 | 全称 | 中文 | 用途 |
|------|------|------|------|
| `regs` | registers | 寄存器 | 通用寄存器数组 |
| `pc` | program counter | 程序计数器 | 指向当前执行指令的地址 |
| `instr` | instruction | 指令 | 从内存取出的 32 位指令编码 |
| `imm` | immediate | 立即数 | 嵌在指令里的常数 |
| `opcode` | operation code | 操作码 | 指令最低 7 位，区分指令大类 |
| `addr` | address | 地址 | 内存地址 |
| `buf` | buffer | 缓冲区 | 数据缓冲区 |
| `sz` | size | 大小 | 数据大小（字节数） |
| `len` | length | 长度 | 同 size，用于字符串或数组长度 |
| `ptr` | pointer | 指针 | 指向某块内存的指针 |
| `sim` | simulator | 模拟器 | Simulator 结构体指针 |
| `d` | decoded | 已解码的 | DecodedInstr 结构体 |
| `r` | register | 寄存器 | `sim->cpu.regs` 的快捷引用 |

## CPU 模块专用

| 缩写 | 全称 | 中文 | 用途 |
|------|------|------|------|
| `rd` | destination register | 目标寄存器 | 指令的 [11:7] 字段 |
| `rs1` | source register 1 | 源寄存器1 | 指令的 [19:15] 字段 |
| `rs2` | source register 2 | 源寄存器2 | 指令的 [24:20] 字段 |
| `f3` / `funct3` | function 3 | 3 位功能码 | 指令的 [14:12] 字段 |
| `f7` / `funct7` | function 7 | 7 位功能码 | 指令的 [31:25] 字段 |

## Memory 模块专用

| 缩写 | 全称 | 中文 | 用途 |
|------|------|------|------|
| `mmu` | memory management unit | 内存管理单元 | 管理虚拟地址翻译和 MemRegion |
| `prot` | protection | 权限位 | 内存区域的读写执行权限掩码 |
| `vpn` | virtual page number | 虚拟页号 | Sv32 页表中的虚拟页号 |
| `pte` | page table entry | 页表项 | Sv32 页表中的一个条目 |

## Debugger 模块专用

| 缩写 | 全称 | 中文 | 用途 |
|------|------|------|------|
| `bp` | breakpoint | 断点 | 程序暂停执行的位置 |
| `repl` | read-eval-print loop | 交互式命令行 | 调试器的命令输入循环 |
| `rsp` | remote serial protocol | 远程串行协议 | GDB 远程调试协议（选做） |

## Loader 模块专用

| 缩写 | 全称 | 中文 | 用途 |
|------|------|------|------|
| `elf` | executable and linkable format | ELF 格式 | 可执行文件格式 |
| `phdr` | program header | 程序头 | ELF 文件的段描述表 |
| `entry` | entry point | 入口地址 | 程序启动时跳转到的地址 |

---

## 规则

1. **用缩写，不用完整单词**：`instr_count` 而非 `instruction_count`。简短不累眼。
2. **RISC-V 官方手册的命名优先**：`opcode`、`funct3`、`rd`、`imm` 直接取 RISC-V 规范中的术语。
3. **新缩写先加到这个文档**：避免各自发明不同的缩写指同一个东西。
